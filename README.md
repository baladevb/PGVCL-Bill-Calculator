# PGVCL Bill Calculator

A simple web-based calculator for estimating electricity bills under PGVCL (Paschim Gujarat Vij Company Limited) tariffs.

## 🌐 Live Demo

Access the calculator online via GitHub Pages:

➡️ [PGVCL Bill Calculator (GitHub Pages)](https://baladevb.github.io/PGVCL-Bill-Calculator/)


## Features
- Supports LT, HT, Residential, Commercial & Agriculture
- Based on GERC tariff (Apr 2026)
- Detailed charge breakdown
- TOU, PF, FCA, ED included
- Prepaid meter rebate option
- Clean UI + print support

## Usage

1. **Online:** Open the [GitHub Pages link](https://baladevb.github.io/PGVCL-Bill-Calculator/) in your browser.
2. **Offline:** Download or clone this repository and open `index.html` in your browser.

## Disclaimer
For estimation only, not official billing. Always refer to official PGVCL sources for actual bills.
