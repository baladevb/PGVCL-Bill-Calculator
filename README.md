# PGVCL-Bill-Calculator
Simple web-based calculator for estimating electricity bills under PGVCL (Paschim Gujarat Vij Company Limited) tariffs
## Features
- Supports LT, HT, Residential, Commercial & Agriculture
- Based on GERC tariff (Apr 2025)
- Detailed charge breakdown
- TOU, PF, FCA, ED included
- Prepaid meter rebate option
- Clean UI + print support

## Usage
Open the HTML file in browser and enter values.

## Disclaimer
For estimation only, not official billing.
